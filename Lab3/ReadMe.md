Lab 3 – Image Processing
This lab focuses on applying fundamental image‑processing operations using Python and OpenCV. The tasks include reading images, converting color spaces, applying filters, and saving processed outputs.

Contents
Grayscale conversion

Color transformations (brown/sepia, green tint, etc.)

Image resizing and scaling

Rotation and flipping

Saving processed images

Displaying results using cv2.imshow()

Objectives
Understand how to manipulate images using OpenCV

Practice applying color filters and transformations

Learn how to save and organize processed outputs

Build confidence with basic image‑processing workflows
